# 📘 Assignment: Building REST APIs with FastAPI

## 🎯 Objective

Learn how to build and deploy RESTful APIs using the FastAPI framework in Python. Practice designing endpoints, handling requests, and returning JSON responses.

## 📝 Tasks

### 🛠️ Create a Simple REST API

#### Description
Build a REST API for a basic resource (e.g., books, students, or tasks) using FastAPI. Implement endpoints for creating, reading, updating, and deleting items.

#### Requirements
Completed program should:

- Use FastAPI to define API endpoints
- Support CRUD operations (Create, Read, Update, Delete)
- Return JSON responses for all endpoints
- Validate input data and handle errors gracefully
- Run locally and be accessible via HTTP requests

### 🛠️ Bonus Task: Add Authentication

#### Description
Add simple authentication (e.g., API key or token) to protect certain endpoints.

#### Requirements
- Require authentication for update and delete operations
- Return appropriate error messages for unauthorized access
